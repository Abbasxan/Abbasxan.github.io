Mən Kiməm?

Mən Sultan — proqramlaşdırma ilə nəfəs alan, texnologiya ilə yaşayan bir gəncəm. Python üzrə ixtisaslaşmışam və əsasən Telegram botları, vebscraping, data analizi və avtomatlaşdırma sahələrində fəaliyyət göstərirəm.

Mənim əsas məqsədim – şirkətlərin və fərdi şəxslərin gündəlik texnoloji proseslərini daha sürətli, səmərəli və etibarlı hala gətirməkdir. Layihələrim arasında bir neçə min istifadəçisi olan Telegram botları, məlumat toplama sistemləri və analitika platformaları var.

İxtisas sahələrimə aşağıdakılar daxildir:

  • 🔹 Telegram botlarının hazırlanması (Pyrogram, Aiogram, Telethon)
  • 🔹 API inteqrasiyası və backend xidmətlər
  • 🔹 MongoDB, PostgreSQL kimi verilənlər bazaları ilə iş
  • 🔹 Data scraping, təmizləmə və vizuallaşdırma
  • 🔹 Linux serverlərin idarə olunması və deploy prosesləri
  • 🔹 Açıq mənbə layihələrdə iştirak və töhfə vermə

Texnologiyaya olan marağım məni daim öyrənməyə və özümü inkişaf etdirməyə vadar edir. Hər gün yeni nəsə öyrənmək, onu tətbiq etmək və başqaları ilə paylaşmaq mənim üçün həm zövq, həm də həyat tərzidir.

Əgər sən də texnologiyanı sevirsənsə, ideyan varsa və ya komanda yoldaşı axtarırsansa — mənimlə əlaqə saxla. Birlikdə böyük işlər görə bilərik!
